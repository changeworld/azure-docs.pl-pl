## <a name="storsimple-and-csp-model"></a>Model zainteresowanie rozwiązaniem StorSimple i dostawcy usług Kryptograficznych

### <a name="what-does-it-mean-to-have-storsimple-as-a-part-of-the-cloud-solutions-provider-csp-program"></a>Co oznacza mają StorSimple w ramach programu dostawca rozwiązań chmury (CSP)?
 Usługa StorSimple dla dostawców CSP umożliwia nasi partnerzy programu CSP do odsprzedaży platformy rozwiązania StorSimple Virtual Array swoim klientom. Ten model umożliwia partnerzy programu CSP, być właścicielem cyklu życia end-to-end klienta za pomocą bezpośredniego inicjowania obsługi administracyjnej, rozliczeń i pomocy technicznej usług w chmurze firmy Microsoft. 

Aby uzyskać więcej informacji, przejdź do [StorSimple w programie CSP](https://www.microsoft.com/en-us/server-cloud/products/storsimple/). 

### <a name="is-storsimple-a-usage-based-or-based-on-seat-licenses-like-office-365-and-enterprise-mobility-suite-ems"></a>Jest StorSimple, a na podstawie użycia lub w zależności od licencji stanowisko, takich jak Office 365 i Enterprise Mobility Suite (EMS)?
StorSimple to usługa oparta na użyciu. Klienci, którzy nabywania rozwiązania StorSimple Virtual Array za pomocą trasy dostawcy usług Kryptograficznych opłata za użycie macierz wirtualną i platformy Azure, które są rozliczane jako oddzielne pozycje na rachunku, dostawcy usług Kryptograficznych. 

### <a name="is-the-storsimple-business-model-similar-to-that-of-other-services-such-as-the-office-365-and-the-enterprise-mobility-suite"></a>Model biznesowy StorSimple jest podobne do innych usług, takich jak Office 365 i pakiet Enterprise Mobility Suite? 
Tak. Model jest modelem Rabat hurtowy, podobnie jak inne na podstawie użycia usługi na platformie Azure. Całościowego rabatu dla urządzenia StorSimple jest podobne do innych usług platformy Azure. Ten model umożliwia nasi partnerzy programu CSP zwiększyć sprzedaż i Zwiększaj użycie platformy Azure za pomocą rozwiązania StorSimple Virtual Array.  

### <a name="which-storsimple-skus-are-available-at-launch"></a>Które StorSimple jednostek SKU są dostępne podczas uruchamiania?
Te same jednostki magazynowe, które są dostępne z autonomicznej usługi StorSimple. 

### <a name="is-there-any-difference-between-the-storsimple-directly-purchased-from-microsoft-under-ea-and-storsimple-as-a-part-of-csp"></a>Czy istnieją różnice między StorSimple zakupione bezpośrednio od firmy Microsoft w ramach umowy EA i urządzeniu StorSimple w ramach programu CSP?
Nie. W obszarze dostawcy usług Kryptograficznych klienci mogą nabyć innych usług oferowanych przez partnera CSP, wraz z usługi StorSimple wirtualnego tablicy (SVA) w ramach jednego faktury. Należy pamiętać, dla nabywanie urządzenia fizycznego StorSimple 8000 series, klienci nadal muszą przechodzić przez tras atrybutów Rozszerzonych. 

### <a name="in-which-markets-is-storsimple-available-for-csp-at-launch"></a>W którym rynków jest StorSimple dostępna dla dostawcy usług Kryptograficznych podczas uruchamiania?
Macierz wirtualna StorSimple dla dostawców CSP jest dostępna na wszystkich rynkach, gdzie StorSimple jest dostępny dzisiaj. Aby uzyskać więcej informacji, przejdź do [listę regionów, w którym usługa StorSimple jest dostępna](https://azure.microsoft.com/regions/services/).

### <a name="what-kind-of-solutions-can-a-partner-deliver-with-storsimple-and-csp"></a>Jakiego rodzaju rozwiązania partnera dostarczać z rozwiązaniem StorSimple i dostawcy usług Kryptograficznych? 
Istnieje kilka rozwiązań:

- Partnerzy mogą prowadzić odsprzedaż rozwiązań StorSimple Virtual Array. 
- Partnerzy mogą dostarczać rozwiązania korzystające z usługi StorSimple i innych usług platformy Azure lub oprogramowania innych firm. Aby uzyskać więcej informacji, przejrzyj kilka możliwych [rozwiązania, które istnieją w całym StorSimple](https://aka.ms/storsimple-build-solutions).
- Mogą też odnajdywać nowych modeli biznesowych przy dostarczaniu usług zarządzanych.

### <a name="what-are-the-incentives-available-for-storsimple-csp-partners-do-i-qualify-for-any-csp-program-incentives"></a>Jakie są dostępne dla partnerzy programu CSP StorSimple zachęty? Czy mogę zakwalifikować się do dowolnej zachęty programu CSP?
Aby uzyskać informacji na temat zachęty, przejdź do [zachęty programu CSP](https://partner.microsoft.com/en-US/cloud-solution-provider/program-details).


## <a name="deploy-and-manage-storsimple-as-a-partner"></a>Wdrażanie i zarządzanie nimi StorSimple jako partnera

### <a name="how-can-i-administer-storsimple-virtual-array-in-csp"></a>Jak można administrować macierzy wirtualnej StorSimple w programie CSP? 
Subskrypcje macierz wirtualną StorSimple można dodać do konta klientów za pośrednictwem Centrum partnerskiego. Ponadto można użyć witryny Azure portal, aby dodać użytkowników do subskrypcji.

### <a name="is-the-azure-portal-approach-the-same-as-other-services-for-csp"></a>Jest taka sama jak innych usług dla dostawcy usług Kryptograficznych podejście do witryny Azure portal? 
Tak. Z rozwiązania StorSimple Virtual Array dla dostawcy usług Kryptograficznych, najlepszym sposobem dla partnerów uzyskać dostęp do portalu Azure należy się za pośrednictwem [Centrum partnerskiego](https://partnercenter.microsoft.com/) której można zarządzać, klientów i subskrypcji. Aby zarządzać subskrypcjami StorSimple, partnera lub klienta (w zależności od uprawnień przyznanych) powinien Zaloguj się do witryny Azure portal. 

### <a name="is-microsoft-shipping-a-new-portal-for-storsimple-for-csp"></a>Microsoft dostarcza nowy portal dla usługi StorSimple dla dostawców CSP? 
Nie. Będzie można administrować macierzy wirtualnej StorSimple w programie CSP za pośrednictwem witryny Azure portal.

### <a name="i-have-provisioned-a-standalone-storsimple-subscription-for-my-customer-do-i-have-to-use-the-azure-portal-to-administer-my-device"></a>Czy mogę dostarczonych autonomiczną subskrypcję usługi StorSimple dla moich klientów. Czy muszę używać witryny Azure portal do administrowania urządzeniem? 
Tak. 

### <a name="is-the-microsoft-account-team-compensated-as-a-result-of-sales-of-storsimple-through-csp"></a>Zespół Microsoft Account jest płatne wyniku sprzedaży usługi StorSimple w ramach programu CSP?
Tak. Dyrektorzy sprzedaży partnera i przez zespół kont klientów dla klienta końcowego będą kompensowane sprzedaży dostawcy usług Kryptograficznych. Partnerzy są szacowany czas potencjalnych klientów w ich wartość zróżnicowanych i ofert sprzedaży.
