# <a name="azure-storage-compliance-offerings"></a>Ofert zgodności w usłudze Azure Storage

Aby ułatwić organizacjom, które są zgodne z krajowe, regionalne i branżowe wymagania dotyczące zbierania i używania danych poszczególnych osób, Microsoft Azure i usługi Azure Storage oferują najbardziej wszechstronny zestaw certyfikatów i atestacji dowolnego Dostawca usług w chmurze.

Można znaleźć poniżej ofert zgodności w usłudze Azure Storage, aby upewnić się, usługa regulowanych przy użyciu usługi Azure Storage. Są one stosowane do następujących ofert usługi Azure Storage: Obiektów blob, plików, kolejek, tabel, dysków, magazynu chłodnego i magazynu w warstwie Premium.
