# <a name="azure-application-contacts-tab"></a>Karta kontaktów w aplikacji platformy Azure

Użyj **kontakty** kartę, aby zapewnić, inżynieria i klienta, informacje dotyczące pomocy technicznej dla usługi Dynamics 365 Customer Engagement oferty.

![Karta kontakty dla oferty](./media/dynce-contacts-tab.png)


## <a name="specify-customer-and-engineering-contacts"></a>Określ odbiorców i inżynieria

Podaj następujące wartości do definiowania podstawowego **kontakt techniczny** i oferty.

|    Pole        |  Opis                    |
|    ---------    |  ---------------                |
|  **Nazwa**       |  Nazwa kontakt techniczny dla aplikacji. Kontakt otrzyma komunikacji techniczne firmy Microsoft.   |
|  **Wiadomość e-mail**      |  Adres e-mail kontakt techniczny  |
|  **Telefon**      |  Numer telefonu kontakt techniczny  |
|  |  |

Podaj następujące wartości do definiowania podstawowego **pomocy technicznej, skontaktuj się z pomocą** i oferty.

|    Pole        |   Opis                   |
|    ---------    |  ---------------                |
|  **Nazwa**       |  Nazwa kontaktu Obsługa klienta dla aplikacji. Kontakt będą otrzymywać informacji dotyczących pomocy technicznej firmy Microsoft.   |
|  **Wiadomość e-mail**      |  Adres e-mail, skontaktuj się z pomocy technicznej dla klientów  |
|  **Telefon**      |  Numer telefonu, skontaktuj się z pomocy technicznej dla klientów  |
|  |  |
