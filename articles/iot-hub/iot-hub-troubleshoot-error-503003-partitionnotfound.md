# <a name="503003-partitionnotfound"></a>503003 PartitionNotFound

W tym artykule opisano przyczyny i rozwiązania **503003 błędów PartitionNotFound** .

## <a name="symptoms"></a>Objawy

Żądania do IoT Hub kończą się niepowodzeniem z błędem **503003 PartitionNotFound**.

## <a name="cause"></a>Przyczyna

Ten błąd jest wewnętrzny dla IoT Hub i prawdopodobnie jest przejściowy. Zobacz, [IoT Hub Przyczyna błędu wewnętrznego serwera](iot-hub-troubleshoot-error-500xxx-internal-errors.md#cause).

## <a name="solution"></a>Rozwiązanie

Zobacz [rozwiązania, na których IoT Hub wewnętrzne błędy serwera](iot-hub-troubleshoot-error-500xxx-internal-errors.md#solution).
