# <a name="quotas-and-limits-for-azure-container-instances"></a>Przydziały i limity dla wystąpień kontenerów platformy Azure

Wszystkich usług platformy Azure dotyczą określone limity i przydziały dla zasobów i funkcji. W tym artykule opisano domyślne przydziały i limity dla wystąpień kontenerów platformy Azure.

Dostępność zasobów obliczeniowych, pamięci i magazynu dla wystąpień kontenerów platformy Azure zależy od regionu i systemu operacyjnego. Aby uzyskać szczegółowe informacje, zobacz [Dostępność zasobów dla wystąpień kontenerów platformy Azure.](container-instances-region-availability.md)
