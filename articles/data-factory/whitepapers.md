# <a name="azure-data-factory-whitepapers"></a>Białe dokumenty usługi Azure Data Factory 

Białe dokumenty umożliwiają eksplorowanie usługi Azure Data Factory na głębszym poziomie. W tym artykule przedstawiono listę dostępnych oficjalnych artykułów dla usługi Azure Data Factory.

| **Whitepaper** | **Opis** |
| --- | --- |
|[Fabryka danych platformy Azure — integracja danych w chmurze](https://azure.microsoft.com/mediahandler/files/resourcefiles/azure-data-factory-data-integration-in-the-cloud/Azure_Data_Factory_Data_Integration_in_the_Cloud.pdf) | W tym artykule opisano, jak usługa Azure Data Factory może umożliwić tworzenie nowoczesnego magazynu danych, umożliwić zaawansowaną analizę w celu obsługi inteligentnych aplikacji SaaS i podnieść pakiety usług integracji programu SQL Server na platformę Azure.|
|[Migracja danych z lokalnego relacyjnego magazynu danych na platformę Azure przy użyciu usługi Azure Data Factory](https://azure.microsoft.com/resources/data-migration-from-on-premise-relational-data-warehouse-to-azure-data-lake-using-azure-data-factory/) | Ten dokument dotyczy złożoności migracji dziesiątek danych TB z istniejącego lokalnego relacyjnego magazynu danych (na przykład Netezza, Oracle, Teradata, SQL server) na platformę Azure (na przykład usługi Blob Storage lub Usługi Azure Data Lake Storage) przy użyciu usługi Azure Data Factory. Wyzwania i najlepsze rozwiązania są zilustrowane w zakresie odporności, wydajności, skalowalności, zarządzania i zabezpieczeń w przypadku podróży pozyskiwania dużych zbiorów danych na platformie Azure przez usługę Azure Data Factory. |
|[Integracja danych SAP przy użyciu usługi Azure Data Factory](https://github.com/Azure/Azure-DataFactory/blob/master/whitepaper/SAP%20Data%20Integration%20using%20Azure%20Data%20Factory.pdf) | W tym artykule podsumowano bieżącą obsługę usługi Azure Data Factory w zakresie integracji danych SAP, w tym scenariusz docelowy, opcje łącznika SAP i porównanie różnych wymagań oraz wprowadzenie do każdego łącznika SAP w fabryce danych. | 
|[Fabryka danych platformy Azure: SSIS w chmurze](https://azure.microsoft.com/mediahandler/files/resourcefiles/azure-data-factory-ssis-in-the-cloud/Azure_Data_Factory_SSIS_in_the_Cloud.pdf)| W tym artykule o wiele więcej niż w przypadku migracji istniejących obciążeń SSIS do usługi Azure Data Factory i rozwiązania typowych zagadnień i problemów. Następnie przeprowadzimy Cię przez szczegóły techniczne tworzenia usługi Azure-SSIS IR, a następnie pokażemy, jak przekazać, wykonać i monitorować pakiety za pośrednictwem usługi Azure Data Factory przy użyciu narzędzi, które prawdopodobnie znasz, takich jak SQL Server Management Studio (SSMS). |
|[Fabryka danych platformy Azure — parametry przekazywania](https://azure.microsoft.com/mediahandler/files/resourcefiles/azure-data-factory-passing-parameters/Azure%20data%20Factory-Whitepaper-PassingParameters.pdf)| W tym artykule opatrzów, jak przekazać parametry między potoku i działania, jak również między działaniami. |
