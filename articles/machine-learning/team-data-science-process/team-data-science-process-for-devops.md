# <a name="team-data-science-process-for-developer-operations"></a>Proces nauki danych zespołu dla operacji deweloperskich

W tym artykule przedstawiono funkcje deweloperskie (DevOps), które są specyficzne dla implementacji zaawansowanej analizy i rozwiązania Cognitive Services. Te materiały szkoleniowe są powiązane z zespołem ds. analizy danych (przetwarzania TDSP) oraz oprogramowaniem i zestawami narzędzi firmy Microsoft i typu open source, które są przydatne w przypadku planowanie, wykonywania i dostarczania rozwiązań do nauki o danych. Odwołuje się do tematów obejmujących DevOps łańcucha narzędzi, które są specyficzne dla projektów i rozwiązań z zakresu nauki i przetwarzania danych.
