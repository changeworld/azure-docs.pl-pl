# <a name="team-data-science-process-for-data-scientists"></a>Proces nauki o danych zespołu dla analityków danych

Ten artykuł zawiera wskazówki dotyczące zestawu celów, które są zwykle używane do implementowania kompleksowych rozwiązań do nauki o danych za pomocą technologii platformy Azure. Jesteś prowadzony przez:

- zrozumienie obciążenia analitycznego
- korzystanie z procesu nauki o danych zespołu
- korzystanie z usługi Azure Machine Learning
- podstawy transferu i przechowywania danych
- dostarczanie dokumentacji źródła danych
- korzystanie z narzędzi do przetwarzania analiz

Te materiały szkoleniowe są związane z team data science process (TDSP) i Microsoft oraz oprogramowaniem i zestawami narzędzi typu open source, które są pomocne w przewidywaniu, wykonywaniu i dostarczaniu rozwiązań do nauki o danych.
