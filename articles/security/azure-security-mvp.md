# <a name="azure-security-mvp-program"></a>Program MVP zabezpieczeń platformy Azure

Microsoft najbardziej specjalistami (MVP) są przywódcami społeczności, którzy mają jawne odszkodowania zaangażowanie w Pomoc innych użytkowników. MVP umożliwić innym najlepiej wykorzystać ich obsługi technologii firmy Microsoft. Współużytkują one ich pasją praktycznej wiedzy i wiedzę techniczną ze społecznością i firmą Microsoft.

Microsoft Azure rozpoznaje teraz ekspertami ze społeczności z wiedzą specjalistyczną w zabezpieczenia platformy Azure. MVPs firmy Microsoft można przyznać MVP w systemie Microsoft Azure, w obszarze wkład zabezpieczeń platformy Azure.

![Microsoft MVP](./media/azure-security-mvp/azure-security-mvp-fig1.png)

Nie ma żadnych testów porównawczych dla staje się MVP. Jest to w części, ponieważ zależy to od technologii i cyklu życia. Obejmuje niektóre kryteria:

- Wpływ kandydat wkład do forów online, np. Microsoft Answers, TechNet i MSDN
- Strony typu wiki i zawartości w trybie online
- Konferencje i grup użytkowników
- Transmisje podcast, witryn sieci Web, blogów i mediów społecznościowych
- Artykuły i książek.
