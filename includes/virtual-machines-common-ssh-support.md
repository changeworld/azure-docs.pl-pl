## <a name="supported-ssh-key-formats"></a>Obsługiwane formaty kluczy SSH

Platforma Azure obsługuje obecnie pary kluczy publiczno-prywatnych protokołu SSH 2 (SSH-2) o minimalnej długości 2048 bitów. Inne kluczowe formaty, takie jak ED25519 i ECDSA, nie są obsługiwane.
