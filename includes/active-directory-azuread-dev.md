> [!IMPORTANT]
> [Microsoft Identity platform (v 2.0)](/azure/active-directory/develop/v2-overview) to ewolucja platformy dla deweloperów Azure Active Directory (Azure AD) (wersja 1.0). Dzięki niej deweloperzy mogą tworzyć aplikacje, które logują użytkowników przy użyciu wszystkich tożsamości firmy Microsoft i uzyskują tokeny do wywoływania interfejsów API firmy Microsoft, takich jak Microsoft Graph, lub interfejsów API opracowanych przez deweloperów.
> Ta zawartość jest dla starszego punktu końcowego usługi Azure AD v 1.0. Zalecamy używanie punktu końcowego v 2.0 w przypadku nowych projektów. Aby uzyskać więcej informacji, przeczytaj artykuł [Dlaczego warto zaktualizować platformę tożsamości firmy Microsoft (wersja 2.0)?](/azure/active-directory/azuread-dev/azure-ad-endpoint-comparison) a także [ograniczenia dotyczące platformy tożsamości firmy Microsoft](/azure/active-directory/azuread-dev/azure-ad-endpoint-comparison#limitations).
